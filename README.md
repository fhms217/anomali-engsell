# Cara Menjalankan dengan TERMUX
1. Update & Upgrade Termux
```
pkg update && pkg upgrade -y
```
2. Install Git
```
pkg install git -y
```
3. Clone repository ini
```
git clone https://github.com/fhms217/anomali-engsell
```
4. Masuk ke folder
```
cd anomali-engsell
```
5. Jalankan setup
```
bash setup.sh
```
6. Jalankan script
```
python main.py
```
7. Masukkan API Key Anda saat diminta
